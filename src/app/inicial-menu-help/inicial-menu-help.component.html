O que é o Science Memory?

O Science Memory é um jogo de memória temático feito especialmente para o Diogo.

Este jogo contém quatro categorias distintas, Espaço, Lego, Avião e Dinossauro. Todas elas a pensar nos gostos do pequeno Diogo.

O jogo é muito simples de jogar, no menu inicial existem quatro imagens representantes de cada uma das categorias. Ao clicar numa das imagens o utilizador é transportado para outra página, onde pode começar a jogar. Cada jogo tem 12 cartas, ou seja, 6 imagens diferentes alusivas ao tema escolhido. O utilizador deve selecionar as cartas para poder ver a sua face de modo a descobrir o seu par e terminar o jogo.

Por último, além de divertido, este jogo tem como objetivo ser educativo, pelo que foram incorporados vídeos educativos sobre cada um dos temas.
